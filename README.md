### Getting Started with Docker Compose

#### Build

```
DOCKER_BUILDKIT=1 COMPOSE_DOCKER_CLI_BUILD=1 docker-compose build
```

#### Launch

```
docker-compose up -d
```

#### Down

```
docker-compose down
```
